# Magento/slate v1.0.0
## Magento 2 slate theme (theme skeleton), based on Magento/blank
### The purpose of this theme is to provide basic skeleton of a theme. A develper can further add/modify files

## Features:
1. Easy to install & use
2. Open Source

___________________________________________________________________________________________________

## Installation:
1. Go to Magento2 root folder

2. Create a directory **app/design/frontend/Magento/slate**

3. Enter following commands to enable module:

    ```bash
    php bin/magento setup:upgrade
    php bin/magento setup:static-content:deploy -f
    ```
4. Enable this theme in Magento Admin under **Content/Design/Configuration**
